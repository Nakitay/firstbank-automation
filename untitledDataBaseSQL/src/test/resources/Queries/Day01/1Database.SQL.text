------------------What is Data?--------------------

All data needs to be stored somewhere, where is secure easy to read, fast to read, easy to update.

------------------What is Database -----------------
Systematic collection of data
storage and manipulation of data
Fast
Two type of Database

1) Relational Database
PK -> Primary key : It is a column that makes each row unique
Can Not Be NULL; Can not be duplicate

FK -> Foreign Key : It is a column that makes connection between tables;
Can be Null;
Can have a Duplicates;

We ise Oracle, MySQL, PostgreSQL;

2) Non relational Database. We dont use it. All Data are in key value format
